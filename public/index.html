Documentation

Currency Converter is a small php application, developed for any person concerned that wants to see my code's quality.

There are a lot of ways to build such an easy small project. It could be developed using just html and javascript, but I have this project as a demo for my PHP knowledge.

This Application has two parts: Server and Client Part.

Server Side makes available this service through 2 endpoints. Format is json.

  • Converter: https://currencyconverter.xman.gr/Server/converter.php
  • Publish available currencies: https://currencyconverter.xman.gr/Server/available_currencies.php

Client sends its requests to Server using PHP. The usage of PHP happens because this is a hypothetical scenario where the client is under a different domain than the Service and I wouldn't like to make cross-domain ajax connections.
When page loads, it connects to Server, for to take the available currencies and put them into the select menus. Then, when user wants a currency conversion, client posts data to server and then show the result.
